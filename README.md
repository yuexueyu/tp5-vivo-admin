# About

此后台是 tp5.1 + mysql 构建的后台管理系统，是后台项目[vivo-shop](https://github.com/Mynameisfwk/vivo-shop) 的管理系统，所有的数据都是从服务器实时获取的真实数据，具有真实数据等功能。
